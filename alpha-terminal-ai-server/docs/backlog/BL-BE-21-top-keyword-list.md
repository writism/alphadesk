# BL-BE-21: 상위 키워드 목록 생성

## 개요

시스템이 빈도순 정렬된 명사 목록에서 상위 N개 키워드를 선택하여 워드클라우드 등 시각화에 활용 가능한 형태로 반환한다.

## Success Criteria

- 시스템은 빈도순 정렬된 명사 목록에서 상위 N개 키워드만 선택한다
- 상위 키워드 목록은 워드클라우드 등 시각화에 활용할 수 있는 형태로 반환된다
- 키워드 개수(N)는 요청 시 파라미터로 조절할 수 있다

## 구현 상태

BL-BE-20의 `GET /youtube/nouns/extract?top_n=30` 엔드포인트에 이미 포함.
본 백로그에서는 응답에 `total_nouns` (전체 추출된 명사 종류 수) 필드를 추가한다.

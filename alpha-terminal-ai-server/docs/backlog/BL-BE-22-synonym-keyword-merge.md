# BL-BE-22: 유사 키워드 동의어 통합

## 개요

시스템이 명사 추출 결과에서 동의어/유사어를 하나의 대표 키워드로 통합하고 빈도수를 합산한다.

## Success Criteria

- 시스템은 동의어 및 유사어를 하나의 대표 키워드로 통합한다 (예: "방산주", "방산株", "방산 관련주" → "방산주")
- 시스템은 사전 정의된 동의어 매핑 테이블을 기반으로 키워드를 통합한다
- 통합된 키워드는 개별 빈도수를 합산하여 반영한다

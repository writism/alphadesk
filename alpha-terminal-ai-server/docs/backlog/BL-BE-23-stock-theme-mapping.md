# BL-BE-23: 관심종목 테마별 종목 매핑 사전 정의

## 개요

시스템이 내 관심종목의 테마별 종목 매핑을 사전 정의하여 DB에 등록한다.
추후 댓글에서 추출된 키워드와 종목 매핑을 연결하여 관련 종목을 추천하는 데 사용한다.

## 도메인

`stock_theme`

## Success Criteria

- 시스템은 내 관심종목의 테마별로 관련 종목을 매핑한다
- 각 종목에는 종목명, 종목코드, 관련 테마 키워드가 포함된다
- DB에 사전 등록되어 있는 종목을 조회할 수 있어야 한다
- 사전 등록 데이터는 이 백로그에서 시딩된다

## 데이터 형태

```json
[
  {
    "name": "한화에어로스페이스",
    "code": "012450",
    "theme": ["전투기", "미사일"]
  },
  {
    "name": "LIG넥스원",
    "code": "079550",
    "theme": ["미사일", "방공"]
  }
]
```

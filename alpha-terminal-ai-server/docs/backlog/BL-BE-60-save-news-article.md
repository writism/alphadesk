# BL-BE-60: 인증된 사용자가 뉴스 검색 결과 중 관심 기사를 저장한다

## Success Criteria

- 이 작업은 news 도메인에서 진행된다
- 인증된 사용자가 관심 기사 저장 요청을 보낼 수 있다
- 요청 입력 정보는 기사 제목, 내용 요약, 출처, 뉴스 링크, 게시 시간, 본문(raw content)을 포함한다
- 기사 메타데이터(제목, 출처, 링크, 게시 시간, 저장 사용자)는 MySQL에 저장된다
- 기사 본문 및 비정형 원본 데이터는 PostgreSQL JSONB 컬럼에 저장된다
- 동일 사용자가 동일 뉴스 링크를 중복 저장할 수 없으며, 중복 요청 시 명확한 에러가 반환된다
- 저장이 완료되면 저장된 기사 식별자(id)와 저장 시각이 응답으로 반환된다

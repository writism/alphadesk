# BL-BE-59: 시스템이 SERP API 클라이언트 환경을 구성한다

## Success Criteria

- 애플리케이션 실행 시 SERP API 접근에 필요한 설정(API Key, Base URL, 타임아웃)이 환경 변수로부터 로드된다
- SERP API 호출을 위한 HTTP 클라이언트가 공용 모듈로 초기화되고, 쿼리 파라미터와 인증 헤더가 일관된 방식으로 주입된다
- 클라이언트는 정상 응답 시 JSON 결과를 반환하고, 오류 응답 시 명확한 예외를 발생시키며 에러 로그를 남긴다
- 네트워크 오류 및 타임아웃에 대해 재시도 또는 실패 처리 정책이 적용된다
- 클라이언트는 SERP API 제공사(예: SerpAPI, SearchAPI)의 Base URL 교체가 환경 변수 변경만으로 가능하도록 구성된다

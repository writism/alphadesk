# BL-BE-24: 종목별 추천 데이터 생성

## 개요

시스템이 댓글에서 추출된 키워드와 DB에 사전 등록된 종목의 테마를 매칭하여 추천 데이터를 생성한다.

## 도메인

`stock_theme`

## Success Criteria

- 시스템은 댓글에서 추출된 키워드와 DB에 사전 등록된 종목의 테마를 매칭하여 추천 데이터를 생성한다
- 각 종목별로 매칭된 키워드, 관련성 점수가 포함된 추천 결과를 반환한다
- 매칭되는 종목이 없는 경우 빈 리스트를 반환한다

## To-do

- [x] DB에 등록된 종목의 테마와 추출된 키워드를 매칭하는 로직을 구현한다
- [x] 종목별 관련성 점수를 산출하는 기능을 구현한다
- [x] 추천 결과를 관련성 점수 기준으로 정렬하여 반환한다
- [x] 추천 결과 Response DTO를 정의한다
- [x] 추천 데이터 생성 API 엔드포인트를 정의하고 라우터에 등록한다

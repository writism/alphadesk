## Success Criteria

- 메뉴 항목이 가로 방향으로 나열되어 표시된다
- 현재 활성 페이지의 메뉴 항목이 시각적으로 강조 표시된다
- 메뉴 항목에 마우스를 올리면 호버 스타일이 적용된다
- 인증 상태에 따라 로그인/로그아웃 버튼이 올바른 위치에 표시된다
- 모든 디자인 항목은 기존 UI와 별개의 파일로 분리되어 언제든 수정하여 변경이 수월하도록 구성되어야 한다

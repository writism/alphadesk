## 슬라이드 12 — 시연 ② 2단계 개인화 Q&A

### 시연 포인트
- `/market-analysis/ask` 에 질문 입력
- LangChain Tool `get_user_profile` 이 최승호 API 호출
- 같은 질문이라도 사용자별로 다른 응답 반환
- "1단계 데이터 → 2단계 맞춤 응답" 연결고리 시각화
- 데모 영상 자리 있음

### get_user_profile Tool 흐름
```
사용자 질문 입력
    ↓
LangChain이 "이 사람 개인 데이터 필요하다" 판단
    ↓
get_user_profile 도구 자동 호출
    ↓
최승호가 만든 GET /users/{user_id}/profile API 호출
    ↓
관심종목 · 투자성향 데이터 가져옴
    ↓
그 데이터 기반으로 맞춤 응답 생성
```

### 발표 멘트
> *"질문을 입력하면 LangChain이 자동으로 get_user_profile 도구를 호출해서 최승호님이 만든 사용자 프로필 API에서 제 관심종목과 투자성향을 가져옵니다. 그 데이터를 바탕으로 저한테 맞는 답변이 생성되는 거죠."*

## 시연 영상 촬영 가이드

- **촬영 도구**: scrcpy (갤럭시 S24+ 미러링) + macOS Shift+Cmd+5 화면 녹화
- **추천 질문**: "내 관심종목 중 오늘 주목할 종목은?" / "내 투자성향에 맞는 최근 뉴스 요약해줘"
- **시연 순서**:
  1. 홈 또는 /my 페이지 — 관심종목·투자성향 데이터 (5초)
  2. /market-analysis/ask 이동 (3초)
  3. 질문 타이핑 → 전송 (10초)
  4. 스트리밍 응답 출력 (15~20초)
  5. 응답 확인 후 멈춤 (3초)
- **영상 길이**: 40~50초 권장
- **라이브 시연** 병행 가능하나 네트워크 지연 대비 영상 백업 필수

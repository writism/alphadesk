# BL-FE-19

**Backlog Type**
Behavior Backlog

**Backlog Title**
대시보드가 최근 AI 분석 내용을 로그 형태로 최신순 표시한다

**배경**
현재 대시보드는 최신 요약 카드와 직전 파이프라인 실행 결과만 보여 주고,
이전에 수행한 AI 분석 내용이 시간 순으로 누적되어 남지는 않는다.
사용자는 최근에 어떤 종목이 어떤 내용으로 분석되었는지,
가장 최근 분석부터 로그처럼 쌓인 형태로 확인하고 싶어 한다.

**Success Criteria**
- 파이프라인 실행으로 생성된 AI 분석 내용이 로그 항목으로 누적 저장된다
- 대시보드에서 최근 분석 로그를 최신순으로 확인할 수 있다
- 각 로그 항목에 분석 시각, 종목, 요약 내용, 태그, 감성 정보가 표시된다
- 새로 분석을 실행하면 최신 로그가 목록 상단에 추가된다
- 기존 AI 분석 요약 카드와 파이프라인 실행 결과 UI에는 회귀가 없다

**Todo**
1. 백엔드에 최근 AI 분석 로그를 조회하는 API를 추가한다
2. 파이프라인 실행 시 생성된 분석 결과를 로그 레지스트리에 누적한다
3. 프론트 대시보드에서 로그 목록을 조회하는 API 호출을 추가한다
4. 대시보드에 최근 분석 로그 섹션을 구현한다
5. 로그가 없을 때의 빈 상태와 최신순 정렬을 확인한다

## 2. 구현 내용

### 2-1. AI 센티멘트 게이지

**산출 공식:**
```
게이지 점수 = (평균 sentiment_score + 1) / 2 × 100
→ 0~100 범위
```

**구간 정의:**

| 점수 | 레이블 | 색상 |
|------|--------|------|
| 0 ~ 35 | 공포 | 파랑 |
| 36 ~ 64 | 중립 | 회색 |
| 65 ~ 100 | 탐욕 | 초록 |

**공포/중립/탐욕 의미:**
- **공포**: AI 분석이 전반적으로 부정적 → 매도 심리 우세
- **중립**: 긍정·부정 혼조 → 관망 심리
- **탐욕**: AI 분석이 전반적으로 긍정적 → 매수 심리 우세
- ⚠️ "내 관심종목 기반" 라벨 필수 표시 (시장 전체 아님)

### 2-2. 오늘의 알파 기회 Top 3

**선별 기준:**
```
정렬 키 = sentiment_score × confidence (내림차순)
조건   = sentiment_score > 0 (긍정 방향만)
표시   = 상위 3건
```

**카드에 표시할 정보:**
- 종목 심볼 + 이름
- 감성 점수 (`sentiment_score`)
- 신뢰도 (`confidence`)
- AI 방향 뱃지 (↑ / ↓)

### 2-3. 감성 분포 한줄 요약

```
긍정 N건 · 중립 N건 · 부정 N건 (총 N건 분석)
```

# BL-FE-24

**Backlog Type**
UX Backlog

**Backlog Title**
파이프라인 실행 중 대시보드에서 분석 중 상태와 상호작용 제한을 명확히 한다

**배경**
분석이 수십 초 걸릴 수 있는데, 선택 체크박스 등이 그대로 활성화되면 중복 실행이나 혼란이 생길 수 있다.

**Success Criteria**
- 실행 중 안내 배너가 유지 또는 강화된다
- 분석 중에는 관심종목 선택 UI가 비활성화되거나 명확히 사용 불가 상태로 보인다
- 루트 `main`에 `aria-busy` 등 접근성 힌트가 반영된다(가능한 범위)

**Todo**
1. `running` 상태를 관심종목 섹션에 전달해 체크박스·전체 선택을 disabled 처리한다
2. 실행 중 배너 문구를 선택 종목 수와 연동해 명확히 한다
3. `main`에 `aria-busy={running}`을 설정한다

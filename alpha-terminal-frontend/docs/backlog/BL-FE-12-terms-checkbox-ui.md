# BL-FE-12

**Backlog Type**
UI Backlog

**Backlog Title**
약관 동의 페이지가 약관 항목마다 체크박스를 표시한다

**배경**
기존 구현은 개별 체크박스만 있고 전체 동의 체크박스가 없다.
전체 동의 체크박스와 개별 체크박스 간 연동이 필요하다.

**Success Criteria**
- 각 약관 항목 옆에 체크박스가 표시된다
- 전체 동의 체크박스가 표시된다
- 전체 동의 체크박스를 선택하면 모든 개별 체크박스가 선택된다
- 개별 체크박스를 모두 선택하면 전체 동의 체크박스가 자동으로 선택된다
- 개별 체크박스 하나를 해제하면 전체 동의 체크박스가 자동으로 해제된다
- 필수 약관 전체 동의 여부에 따라 동의 버튼 활성화 상태가 표시된다

**Todo**
1. 각 약관 항목에 개별 체크박스를 표시한다
2. 전체 동의 체크박스를 표시한다
3. 전체 동의와 개별 체크박스 간 연동을 구현한다
4. 체크박스 선택 상태를 관리하는 로직을 구현한다
5. 필수 약관 전체 동의 여부에 따라 동의 버튼 활성화 상태를 표시한다

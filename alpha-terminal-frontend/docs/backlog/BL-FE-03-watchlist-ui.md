# BL-FE-03

**Backlog Type**
UI Backlog

**Backlog Title**
관심종목 페이지가 종목 검색 및 등록 UI를 표시한다

**의존성**
- BL-FE-01, BL-FE-02 선행 완료 후 진행

**Success Criteria**
- 종목 코드 또는 종목명을 입력하는 검색 입력창이 표시된다
- 검색 중에는 로딩 상태가 입력창 영역에 표시된다
- 검색 결과가 있으면 종목 코드, 종목명, 시장명이 목록으로 표시된다
- 각 검색 결과 항목에 등록 버튼이 표시된다
- 등록된 관심종목 목록에 종목 코드, 종목명, 시장명이 표시된다
- 중복 등록 시도 시 안내 메시지가 표시된다

**Todo**
1. 종목 검색 입력창과 검색 결과 목록 UI를 구현한다
2. 검색 결과 항목에 종목 코드, 종목명, 시장명과 등록 버튼을 구현한다
3. 등록된 관심종목 목록 UI를 구현한다 (종목 코드, 종목명, 시장명, 삭제 버튼)
4. 검색 로딩 및 빈 결과 상태 UI를 구현한다
5. 중복 등록 에러 메시지 UI를 구현한다

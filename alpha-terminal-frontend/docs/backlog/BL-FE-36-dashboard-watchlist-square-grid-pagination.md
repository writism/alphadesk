# BL-FE-36

**Backlog Type**  
UX / Layout

**Backlog Title**  
대시보드 관심종목 그리드 — 컴팩트 카드 + 페이지네이션

---

## 1. 배경

- 대시보드 상단 **관심종목**은 카드형 그리드로 보여 준다.
- 종목이 많을 때 한 화면에 모두 그리면 **밀도·스크롤**이 커진다.
- `/watchlist` 페이지와 달리 대시보드 그리드에는 아직 페이지네이션이 없다.

## 2. 목표 (Success Criteria)

| ID | 기준 |
|----|------|
| SC-1 | 카드는 **콘텐츠(티커·뱃지·종목명)에 맞는 높이**로 두고, 불필요한 세로 빈 공간을 두지 않는다. (초안의 정사각형은 폐기.) |
| SC-2 | 그리드는 반응형 열 수(2/3/4)를 유지한다. |
| SC-3 | **페이지당 12건**(4열×3행에 맞춤)씩만 렌더하고, **이전/다음** 페이지 바를 둔다(BL-FE-35와 동 패턴). |
| SC-4 | **전체 선택**·`선택 N/M`은 **전체 관심종목** 기준을 유지한다(페이지는 표시만 분리). |
| SC-5 | 종목명이 길면 **말줄임(line-clamp)** 등으로 카드 안에 넘치지 않게 한다. |

## 3. 비범위

- 관심종목 API 페이지네이션.
- `/watchlist` 풀 리스트 레이아웃 변경(별도 화면).

# BL-FE-60: 인증된 사용자가 페이지네이션 된 주식 관련 뉴스 목록을 조회한다

## Success Criteria

- Backend에 요청하는 API는 /news/search 로 요청한다.
- 인증된 사용자가 뉴스 페이지에 진입하면 현재 페이지 번호와 페이지 크기를 포함한 주식 관련 뉴스 목록 요청이 전송된다
- 요청이 성공하면 해당 페이지의 주식 관련 뉴스 목록과 전체 페이지 정보가 응답으로 반환된다
- 응답으로 받은 뉴스 목록이 전역 상태에 저장되어 UI에서 조회 가능한 상태가 된다
- 사용자가 다른 페이지를 선택하면 해당 페이지의 뉴스 목록이 다시 요청되어 상태가 갱신된다
- 인증되지 않은 사용자가 요청할 경우 뉴스 목록은 조회되지 않고 인증 실패 상태로 처리된다
- 한 페이지에는 10개의 항목을 보여준다
- 사용자 인증 정보인 user_token은 Cookie 기반으로 전달된다
- 요청하는 정보에 키워드로서 내 관심종목과 관련있는 키워드를 포함시켜 요청한다
- 화면 출력 시 이미지를 표시하는 부분은 필요하지 않다
- 뉴스 리스트 페이지에 있는 뉴스 항목마다 저장 버튼을 표시한다

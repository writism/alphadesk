# BL-FE-01

**Backlog Type**
Behavior Backlog

**Backlog Title**
인증된 사용자가 종목 코드 또는 종목명으로 종목을 검색한다

**의존성**
- 백엔드 `GET /stocks/search?q=` 구현 필요 → BL-BE-01 선행 완료 후 진행

**Success Criteria**
- 종목 코드(예: 005930, AAPL) 또는 종목명(예: 삼성전자, Apple)을 입력하면 검색 요청이 전송된다
- 검색 결과로 종목 코드, 종목명, 시장명(코스피/코스닥/나스닥 등)이 반환된다
- 한글 검색 결과가 0건이면 자동으로 영문으로 변환하여 재검색한다
- 재검색 시 사용자에게 "영문으로 재검색 중" 상태가 표시된다
- 입력값이 1자 미만이면 검색 요청이 전송되지 않는다
- 한글/영문 모두 결과가 없으면 빈 목록 상태가 반환된다
- 검색 API 호출 실패 시 에러 상태가 반환된다

**Todo**
1. 종목 검색 API 호출 로직을 구현한다 (`GET /stocks/search?q=`)
2. 검색 결과가 0건이면 영문 검색어로 변환하여 재검색하는 fallback 로직을 구현한다
3. 재검색 중임을 알리는 상태를 관리한다
4. 검색 결과를 종목 코드, 종목명, 시장명 형태로 가공한다
5. 검색어가 비어있을 때 검색을 방지하는 유효성 검사를 구현한다

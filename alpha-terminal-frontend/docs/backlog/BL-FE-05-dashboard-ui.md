# BL-FE-05

**Backlog Type**
UI Backlog

**Backlog Title**
대시보드 페이지가 관심종목 목록을 표시한다

**의존성**
- BL-FE-04 선행 완료 후 진행

**Success Criteria**
- 등록된 관심종목이 카드 형태로 표시된다
- 각 카드에 종목 코드, 종목명, 시장명이 표시된다
- 로딩 중에는 스켈레톤 또는 로딩 UI가 표시된다
- 등록된 관심종목이 없으면 안내 문구가 표시된다

**Todo**
1. 관심종목 카드 UI를 구현한다 (종목 코드, 종목명, 시장명)
2. 로딩 상태 UI를 구현한다
3. 빈 상태 안내 UI를 구현한다

# BL-FE-07

**Backlog Type**
UI Backlog

**Backlog Title**
대시보드 페이지가 분석 결과 없음 상태를 명확하게 안내한다

**배경**
`GET /pipeline/summaries`가 빈 배열을 반환할 때 현재는 화면에 아무것도 표시되지 않는다.
사용자는 로딩 중인지, 오류인지, 실제로 결과가 없는 것인지 구분할 수 없다.

**Success Criteria**
- 분석 결과가 없을 때 "아직 분석된 종목이 없습니다" 안내 문구가 표시된다
- 안내 문구와 함께 파이프라인 실행 버튼 또는 관심종목 등록 안내가 표시된다
- 로딩 완료 후 빈 결과임을 확실히 구분할 수 있는 UI가 표시된다

**Todo**
1. 분석 결과 없음 빈 상태 UI를 구현한다
2. 빈 상태에서 "파이프라인 실행" 또는 "관심종목 등록" 안내 액션을 표시한다
3. 로딩 상태와 빈 상태를 시각적으로 구분한다

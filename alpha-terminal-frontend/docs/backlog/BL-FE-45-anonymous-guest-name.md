# BL-FE-45 익명 댓글 자동 이름 생성

## 배경
댓글 작성 시 닉네임을 입력하지 않으면 "익명"으로 표시된다.
사용자마다 고유한 이름이 없어 피드에서 구별이 어렵고, 이름 입력 필드가 비어 있어 UX가 어색하다.

## 목표
- 사용자가 닉네임을 입력하지 않아도 "한국어 형용사 + 동물명 + 4자리 숫자" 형식의 이름을 자동 부여한다.
- 생성된 이름은 쿠키(`anon_name`)에 저장해 같은 브라우저에서는 항상 동일한 이름으로 표시된다.
- 로그인한 사용자는 `nickname` 쿠키 값으로 자동 채워진다.

## 구현 범위
- `features/share/infrastructure/utils/guestName.ts` 신규 생성
  - 한국어 형용사 15개 × 동물 15개 × 1000~9999 난수
  - `getOrCreateGuestName()`: 쿠키 확인 → 없으면 생성 후 저장 (1년 만료)
- `CommentModal.tsx`: 닉네임 필드 기본값을 자동 생성 이름으로 채움
  - 로그인 사용자: `nickname` 쿠키 우선 사용

## 완료 기준
- 댓글 모달을 열면 닉네임 필드가 자동 생성 이름으로 채워진다
- 같은 브라우저에서 다시 열어도 동일한 이름이 표시된다
- 사용자가 이름을 직접 수정하고 등록하면 해당 이름으로 댓글이 달린다

# BL-FE-15

**Backlog Type**
Behavior Backlog

**Backlog Title**
약관 동의 페이지가 보기 클릭 시 약관 상세 내용을 표시한다

**배경**
현재 약관 동의 페이지는 각 약관 항목 옆에 `보기` 링크를 표시하지만,
클릭 시 이동할 상세 영역이 없어 실제 약관 본문이 노출되지 않는다.
약관 데이터는 `TERMS_DATA.sections`에 이미 존재하므로,
사용자가 이용약관, 개인정보처리방침, 아동 보호 정책의 내용을 화면에서 확인할 수 있도록
상세 보기 UI를 연결해야 한다.

**Success Criteria**
- 각 약관 항목의 `보기` 클릭 시 해당 약관의 상세 내용이 화면에 표시된다
- 표시되는 상세 내용은 `TERMS_DATA.sections`의 제목과 본문을 그대로 사용한다
- 같은 약관 항목의 `보기`를 다시 누르면 상세 내용이 닫히거나 숨겨진다
- 다른 약관 항목의 `보기`를 눌러도 해당 항목의 상세 내용을 확인할 수 있다
- 약관 상세 내용이 표시되어도 기존 체크박스 동의 흐름은 유지된다
- 필수 약관 동의 여부 계산과 회원가입 이동 흐름에 회귀가 없다

**Todo**
1. 약관 상세 보기 상태를 관리하는 UI state를 `TermsPage`에 추가한다
2. `보기` 클릭 시 선택한 약관 항목의 상세 내용을 토글하는 동작을 구현한다
3. `TERMS_DATA.sections`를 순회하여 조 제목과 본문을 렌더링하는 상세 영역을 구현한다
4. 상세 영역이 체크박스 목록과 겹치지 않도록 레이아웃과 스타일을 정리한다
5. 약관 동의, 전체 동의, `/account/signup` 이동 흐름이 기존처럼 동작하는지 확인한다

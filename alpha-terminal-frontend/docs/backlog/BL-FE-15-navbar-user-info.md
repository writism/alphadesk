# BL-FE-15

**Backlog Type**
UI Backlog

**Backlog Title**
내비게이션 바가 로그인한 사용자 정보를 표시한다

**배경**
현재 Navbar는 로그인/로그아웃 버튼만 표시하고 사용자 정보를 표시하지 않는다.
로그인한 사용자의 닉네임을 Navbar에 표시하여 현재 로그인 상태를 확인할 수 있어야 한다.

**Success Criteria**
- 로그인한 상태에서 Navbar에 사용자 닉네임이 표시된다
- 로그아웃 상태에서는 닉네임이 표시되지 않는다

**Todo**
1. 로그인 상태에서 사용자 닉네임을 Navbar에 표시한다

# BL-FE-20

**Backlog Type**
Behavior Backlog

**Backlog Title**
대시보드가 선택한 관심종목만 AI 분석 실행한다

**배경**
현재 대시보드의 `최신 분석 실행` 버튼은 관심종목 전체를 대상으로 파이프라인을 실행한다.
사용자는 모든 종목을 한 번에 분석하는 대신,
관심종목 중 원하는 종목만 선택해서 개별 또는 부분 분석을 실행하고 싶어 한다.

**Success Criteria**
- 대시보드에서 관심종목별 선택 UI를 확인할 수 있다
- 사용자는 하나 이상의 관심종목을 선택한 뒤 선택한 종목만 AI 분석할 수 있다
- 선택한 종목만 백엔드 파이프라인 대상으로 전달된다
- 분석 실행 결과와 최근 AI 분석 로그에는 선택한 종목 결과만 반영된다
- 아무 종목도 선택하지 않으면 분석 실행 버튼이 비활성화되거나 실행되지 않는다
- 기본 진입 시에는 기존 사용성을 위해 전체 선택 상태로 시작한다

**Todo**
1. 대시보드 관심종목 영역에 종목 선택 UI를 추가한다
2. 선택 심볼 목록을 포함해 파이프라인 실행 API를 호출하도록 수정한다
3. 백엔드 파이프라인이 선택된 종목만 처리하도록 입력 모델을 확장한다
4. 분석 실행 결과와 로그가 선택된 종목만 반영되는지 확인한다

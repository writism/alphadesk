# Alpha Terminal 사용자 시나리오 — 온보딩 & 메뉴 가이드

> 작성일: 2026-04-18  
> 대상: 팀원(개발/기획), 신규 사용자, 기존 사용자  
> 목적: 로그인 후 앱을 어떻게 써야 하는지 첫 흐름을 정의하고, 각 메뉴의 역할을 명확히 공유한다.

---

## 왜 이 문서가 필요한가

기존에는 로그인 후 무엇을 해야 하는지 명확한 안내가 없었다.  
Alpha Terminal은 **관심종목(MY_WATCHLIST) 설정**이 되어야만 브리핑·AI 분석·투자판단 등 핵심 기능이 의미 있게 동작한다.  
사용자가 이를 모른 채 다른 메뉴를 먼저 탐색하면 "왜 내용이 없지?" 하는 혼란을 겪는다.  
이를 해결하기 위해 **MY 페이지에 GETTING_STARTED 온보딩 가이드**가 내장되어 있다.

---

## 메뉴 구조 (TopBar 기준)

```
ALPHA_TERMINAL
├── HOME            /              관심종목 기반 AI 브리핑 + 주요 지표
├── DASHBOARD       /dashboard     관심종목 종합 분석 실행 및 결과 확인
├── MARKET_FEED     /market-feed   시장 정보 피드 (탭 구조)
│   ├── 뉴스_피드                   실시간 주요 금융 뉴스
│   └── 영상_피드                   YouTube 금융 채널 영상 모음
├── AI_INSIGHT      /ai-insight    AI 분석 도구 모음 (탭 구조)
│   ├── 투자_판단                   종목 매수·매도·보유 판단 · 멀티에이전트 심층분석
│   └── 종목_추천                   내 관심종목 기반 AI 맞춤 분석
├── BOARD           /board         커뮤니티 의견 공유
└── MY (우측)       /my            관심종목 관리 + 브리핑 설정 + 온보딩 가이드
```

## 시나리오 1 — 신규 사용자 (최초 로그인)

```
[로그인]
    ↓
[MY 페이지 자동 이동] ← 로그인 완료 후 자동 리다이렉트
  → GETTING_STARTED 온보딩 가이드 표시
     01. MY_WATCHLIST에서 관심 종목 추가
     02. SETTINGS → BRIEFING_TIME 설정
     03. 설정한 시간에 TODAY_BRIEFING 자동 생성
    ↓
[MY_WATCHLIST에서 종목 추가]
  → 예: 삼성전자, 카카오, SK하이닉스
    ↓
[SETTINGS에서 브리핑 수신 시간 설정]
  → 기본값: 오전 7시 (KST)
    ↓
[HOME으로 이동]
  → 설정 시간이 되면 TODAY_BRIEFING 자동 생성
  → 관심종목 기반 AI 분석 요약 확인
    ↓
[DASHBOARD로 이동]
  → "분석 시작" 버튼 → 관심종목 파이프라인 즉시 실행
  → AI 분석 요약 카드, 감성 지표, 리포트 확인
    ↓
[AI_INSIGHT / MARKET_FEED 자유 탐색]
```

**핵심 포인트**  
관심종목이 없으면 HOME·DASHBOARD·AI_INSIGHT 모두 빈 상태가 된다.  
MY 페이지의 GETTING_STARTED 가이드가 이를 안내한다.

## 시나리오 2 — 기존 사용자 (재방문)

```
[로그인]
    ↓
[MY 페이지 자동 이동] ← 로그인 완료 후 자동 리다이렉트
    ↓
[HOME으로 이동]
  → 오늘의 TODAY_BRIEFING 확인 (설정한 시간에 자동 생성)
  → 관심종목 변동 요약, 시장 감성 지표 확인
    ↓
[필요에 따라 세부 탐색]
  ├── 특정 종목 심층 분석 필요
  │     → AI_INSIGHT → 투자_판단 탭
  │     → "삼성전자 지금 사도 될까?" 입력
  │     → 멀티에이전트 실시간 분석 로그 확인 → 최종 판단
  │
  ├── AI 맞춤 종목 추천 확인
  │     → AI_INSIGHT → 종목_추천 탭
  │
  ├── 뉴스·공시 확인
  │     → MARKET_FEED → 뉴스_피드 탭
  │
  ├── YouTube 여론/감성 확인
  │     → MARKET_FEED → 영상_피드 탭
  │
  ├── 전체 분석 결과 재조회
  │     → DASHBOARD → 최근 분석 카드 확인
  │
  └── 관심종목 변경 또는 브리핑 시간 조정
        → MY 페이지
```

```

    ↓

[HOME] (모바일 레이아웃)

  → 브리핑 시간 설정
```

## 메뉴별 상세 설명

### HOME (`/`)
- **목적**: 오늘의 AI 브리핑 + 관심종목 현황 한눈에 보기
- **빈 상태**: 관심종목 미설정 시 "MY에서 관심종목을 추가하세요" 안내
- **브리핑 생성**: MY > SETTINGS > BRIEFING_TIME에서 설정한 시간에 자동 생성

### DASHBOARD (`/dashboard`)
- **목적**: 관심종목 전체에 대한 분석 파이프라인 즉시 실행 + 결과 조회
- **사용법**: "분석 시작" 클릭 → 뉴스/공시/YouTube 수집 → AI 요약 생성
- **결과**: 종목별 분석 요약 카드, 감성 점수, 리포트 요약

### MARKET_FEED (`/market-feed`)
- **뉴스_피드 탭**: 실시간 주요 금융 뉴스 목록
- **영상_피드 탭**: YouTube 금융 채널 영상 + 감성 지표

### AI_INSIGHT (`/ai-insight`)
- **투자_판단 탭**: 자유 질의 → AI 멀티에이전트 실시간 분석 (캐시 1시간)
- **종목_추천 탭**: 관심종목 기반 AI 맞춤 종목 추천

### BOARD (`/board`)
- **목적**: 커뮤니티 의견 공유 및 토론

### MY (`/my`)
- **목적**: 앱의 핵심 설정 허브
- **구성**:
  - `GETTING_STARTED`: 신규 사용자 온보딩 가이드 (관심종목 0개일 때 표시)
  - `MY_WATCHLIST`: 관심종목 추가/삭제
  - `TODAY_BRIEFING`: 오늘 생성된 브리핑 확인
  - `SETTINGS`: 브리핑 수신 시간 설정

| 우선순위 | 항목 | 현황 |
|---|---|---|
| ✅ 완료 | MY 페이지 GETTING_STARTED 가이드 | 구현됨 |
| ✅ 완료 | 관심종목 0개 시 온보딩 가이드 자동 표시 | 구현됨 |
| 🔴 필요 | HOME 빈 상태에서 MY로 유도하는 CTA | 미구현 |
| ✅ 완료 | 로그인 후 MY 페이지로 자동 리다이렉트 (신규/기존 공통) | 구현됨 |

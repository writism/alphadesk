# TEST-01: 애플리케이션이 시작할 때 .env 환경 변수를 로드한다

## Success Criteria

- 애플리케이션 시작 시 .env 파일의 환경 변수가 정상적으로 로드된다
- 필수 환경 변수가 누락된 경우 오류가 발생하고 애플리케이션이 시작되지 않는다
- 환경 변수 값은 코드 내부에서 직접 참조하지 않고 설정 모듈을 통해서만 접근할 수 있다
- 클라이언트 노출 변수와 서버 전용 변수가 올바르게 분리된다
- 실제 관리하는 변수는 `NEXT_PUBLIC_API_BASE_URL`, `NEXT_PUBLIC_KAKAO_LOGIN_PATH` 이다

# BL-FE-10

**Backlog Type**
UI Backlog

**Backlog Title**
약관 동의 페이지가 약관 목록을 표시한다

**배경**
기존 약관 페이지는 단일 스크롤 텍스트 박스와 하나의 체크박스로 구성되어 있어
개별 약관 항목의 필수/선택 구분이 불가능하다.
이용약관, 개인정보처리방침, 아동 보호 정책을 각각 항목으로 분리하여 표시해야 한다.

**Success Criteria**
- 이용약관 항목이 화면에 표시된다
- 개인정보처리방침 약관 항목이 화면에 표시된다
- 아동 보호 정책 약관 항목이 화면에 표시된다
- 각 약관 항목에 필수/선택 라벨이 표시된다
- 각 약관 항목에 상세 내용 확인 링크가 표시된다
- 필수 약관을 모두 동의해야 다음 단계로 진행할 수 있다

**Todo**
1. 약관 항목 목록(이용약관, 개인정보처리방침, 아동 보호 정책)을 화면에 표시한다
2. 각 약관 항목에 필수/선택 라벨을 표시한다
3. 약관 상세 내용을 확인할 수 있는 링크를 표시한다

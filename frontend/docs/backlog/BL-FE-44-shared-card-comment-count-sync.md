# BL-FE-44 공유 카드 댓글 수 실시간 동기화

## 배경
다른 기기(모바일)에서 댓글을 추가해도 웹 화면의 댓글 수는 페이지 로딩 시점의 값을 그대로 유지한다.
카드 목록이 30초마다 자동 갱신되지 않아 댓글 수가 stale 상태로 남는다.

## 목표
- 공유 카드 피드가 30초마다 자동으로 목록을 조용히(loading 스피너 없이) 갱신한다.
- 현재 사용자가 댓글을 추가하면 즉시 피드를 재조회해 카운터를 최신 값으로 반영한다.

## 구현 범위
- `useSharedCards.ts`: 30초 폴링 추가 (silentRefresh — loading 상태 변경 없음)
- `SharedCardFeed.tsx`:
  - `onReload` 콜백을 `SharedCardItem`에 전달
  - 댓글 추가 후 즉시 `reload()` 호출

## 완료 기준
- 타 기기에서 댓글 추가 시 최대 30초 내에 카운터 자동 갱신
- 직접 댓글 추가 시 즉시 카운터 반영

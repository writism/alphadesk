# BL-FE-38

**Backlog Type**  
UX Backlog

**Backlog Title**  
모바일 내비게이션: 로고·메뉴·닉네임·로그아웃을 심플하게 정리한다

**배경**  
좁은 뷰포트에서 상단 바가 한 줄에 모두 배치되면서 로고 줄바꿈, 메뉴 압축, 「닉네임」 레이블 글자 단위 줄바꿈, 로그아웃 버튼 잘림 등이 발생한다.

**Success Criteria**
- `md` 미만: **로고 + (로그인 시 햄버거 / 비로그인 시 Login)** 만 노출되어 한 줄이 과밀하지 않다.
- `md` 미만: Home·Dashboard·Watchlist·사용자 정보·Logout은 **슬라이드 패널(또는 동등한 오버레이 메뉴)** 에서만 노출된다.
- 닉네임·이메일은 **한국어 줄바꿈**이 단어 단위를 깨지 않도록 처리한다 (`break-keep` 등).
- 오버레이 열림 시 스크롤 잠금, 배경 클릭·닫기 버튼·경로 변경 시 패널이 닫힌다.
- 키보드: 햄버거에 `aria-expanded`, 패널에 `aria-modal`·포커스 가능한 닫기 버튼.
- `md` 이상: 기존과 동등하게 가로 메뉴 + 사용자 블록 + Logout이 보인다.

# BL-BE-66: 백엔드 서버가 관심 종목 이슈를 능동적으로 감지하여 알림을 생성한다

> **담당**: 김동녁 | **단계**: 3단계 | **주차**: Week 3~4

## 배경

사용자가 앱을 켜지 않아도 AI가 관심 종목의 중요 이슈(급등락, 공시, 주요 뉴스)를
선제적으로 감지하고 알림을 생성하는 백그라운드 에이전트가 필요하다.

## Success Criteria

- 매일 08:00 KST 스케줄러가 모든 사용자의 관심종목 이슈를 점검한다
- AI 에이전트가 중요도를 판단하여 알림 생성 여부를 결정한다
- 생성된 알림은 `notifications` 테이블에 저장된다 (BL-BE-67 의존)
- 동일 이슈에 대해 중복 알림이 생성되지 않는다
- 에이전트 실행 로그가 기록된다

# BL-BE-58: 인증된 사용자가 뉴스 검색 결과를 페이징 조회한다

## Success Criteria

- 인증된 사용자가 뉴스 검색 요청을 보낼 수 있다
- SERP API를 사용하여 뉴스를 검색한다
- 요청 입력 정보는 검색 키워드, 페이지 번호, 페이지 크기를 포함한다
- 시스템은 검색 키워드를 기준으로 뉴스 데이터를 필터링하고 페이지 단위로 조회한다
- 반환 데이터에는 뉴스 제목, 내용 요약, 출처, 뉴스 링크, 게시 시간, 전체 결과 수가 포함된다
- 검색 결과가 없는 경우 빈 리스트와 전체 결과 수 0이 반환된다
- 기본적으로 한 페이지에 10개의 항목이 존재한다
